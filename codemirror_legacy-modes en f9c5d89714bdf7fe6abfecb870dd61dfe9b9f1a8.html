    Modos @codemirror/heredados

    [ SITIO WEB | PROBLEMAS | FORO | REGISTRO DE CAMBIOS ]

    Este paquete implementa una colección de secuencias portadas Modos de idioma para el editor de código de CodeMirror. Cada modo es disponible como un archivo de script independiente, en , y exporta los valores que se enumeran a continuación."@codemirror/legacy-modes/mode/[name]"

    La página del proyecto tiene más información, un número de ejemplos y la documentación.

    Este código se publica bajo una licencia MIT.

    Nuestro objetivo es ser una comunidad inclusiva y acogedora. Para hacerlo explícito, Tenemos un código de Conducta que aplica a la comunicación en torno al proyecto.

    Ejemplo

    El uso de modos de este paquete funciona de la siguiente manera:

    • Instale este paquete y el paquete @codemirror/stream-parser.

    • Encuentre la instancia que necesita en la referencia a continuación.StreamParser

    • Añade a tu editor configuración.StreamLanguage.define(theParser)

    Por ejemplo, para cargar el modo Lua, harías algo como...

    import {StreamLanguage} from "@codemirror/stream-parser"
import {lua} from "@codemirror/legacy-modes/mode/lua"

import {EditorView, EditorState, basicSetup} from "@codemirror/basic-setup"

let view = new EditorView({
  state: EditorState.create({
    extensions: [basicSetup, StreamLanguage.define(lua)]
  })
})
